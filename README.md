Este projeto é o início de uma jornada para estudo. Nele eu desenvolvi uma página de um blog,  procurei utilizar apenas HTML, CSS e Js por ser uma página simples. Com esse site, estudei Flexbox e responsividade com o CSS, relembrando como adequar a tela para o dispositivo utilizado.
Foi um projeto simples, mas que me trouxe de volta a essência que é desenvolver front-end.
